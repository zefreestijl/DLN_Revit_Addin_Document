Basic Modeling 基本建模: 结构系统, 基本幕墙或外墙, 楼板...等模型类的新建或修改辅助工具 4. Advanced Modeling 进阶建模: 幕墙分隔, 幕墙或墙体, 楼板, 屋顶...等子单元的新建或修改辅助工具 5. Library Browser 族库浏览器: 平, 立, 剖面, 三维等视图类 或 材料, 样式轮廓...等设定类 或 族元件类的列表生成辅助工具 6. Annotation 标示注释: 尺寸, 标签, 符号, 图例, 剖面填充, 门窗图...等标注类生成与更新的辅助工具 7. View & Layout 版面布置: 墙身剖面, 分段剖面, 各向立面...等视图生成和图纸空间布局的辅助工具 8. Analysis 分析模拟: 日照分析, 体量分析, 各专业系统检核的分析与模拟辅助工具 9. Import & Export 导入导出: 导出三维格式, 导入CAD清理, 档案模型加密, 或中心文挡版次比对和修复等各档案间相互转换的辅助工具
